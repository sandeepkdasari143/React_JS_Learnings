->Why `export 'default' FuctionalComponentName`?
	-> Why `default` word?
	-> What are the other ways to export the file.

-> There are two ways to export any React Functional Component, they are:
	
	1. Exporting by Name.
		e.g., `export const Title = ({...props}) => {
					return(
						<Logo />
					);
				};`
	2. Exporting by Default
		e.g., `const Header = () => {
					return(
						<>
							<Title />
							<NavBar />
						</>
					);
				};
				export default Header;`
